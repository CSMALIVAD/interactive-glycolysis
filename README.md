# 🧬 Metabolic Pathway Explorer (Interactive Vertical Mode)

A fully interactive metabolic pathway explorer built using pure HTML, CSS, and Vanilla JavaScript.

Starts from **Glucose** and builds step-by-step vertically.

---

## 🚀 Features

- Structured vertical pathway
- Keyboard-controlled navigation
- Glycolysis
- TCA Cycle
- Gluconeogenesis
- Pentose Phosphate Pathway
- Beta-oxidation
- Carbon tracking
- Energy tracking

---

## ⌨️ Keyboard Controls

| Key | Action |
|-----|--------|
| ENTER | Next step |
| BACKSPACE / ← | Undo |
| R | Reset |
| T | Build TCA |
| P | Pentose Phosphate Pathway |
| B | Beta-oxidation |

---

## 🔬 Included Pathways

- Glycolysis
- TCA Cycle
- Gluconeogenesis
- HMP Shunt
- Beta-Oxidation
- Lactate Pathway
- Alanine Pathway

---

## 🎨 Visual Guide

- Blue = Carbon number
- Green = Produced
- Red = Consumed
- → Irreversible
- ↔ Reversible

---

## ▶️ How to Use

Open `index.html` in your browser and use keyboard controls to explore pathways.
